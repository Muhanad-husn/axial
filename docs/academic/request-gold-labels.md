# Gold-set labels — already in flight (pointer only)

*This deliverable is handled on its own track. It is recorded here so this folder
is the complete picture of what any academic is asked for.*

## What it is

A labelling spreadsheet for the tagging evaluation: roughly 100–120 short passages,
one per row. The academic labels the tag columns so we can measure how well the
automatic tagger agrees with an expert.

- **Label from scratch (blind):** `claim_type`, `theory_school`.
- **Correct our guess (pre-filled):** `field`, `empirical_scope`, `polities_touched`.

## Status

Being sent separately via the offline handoff bundle the pipeline generates
(`axial gold deliver`), which contains the sheet, a `README-for-academic.md` with
the full labelling instructions, and a manifest. That README is the authority on
how to fill the sheet. Nothing here overrides it.

This is usually **one** person's careful pass, not a distributed ask. If that
person is already lined up, there is nothing to do here.
