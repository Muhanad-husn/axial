| DEC-33 | 2026-07-21 | **Abstention is a per-axis flag on the tag record, never a value in the controlled vocabulary — DEC-31's deferred question, settled.** DEC-31 adopted best-of-N on the blind axes and closed with "what the pipeline writes on abstention is an open design question deferred to that issue, not settled here." Issue #294 (PR #302) settles it: when an axis's `N` draws hold no strict plurality, that axis records `{primary: null, abstained: true, draws: [<the distinct primaries, in draw order>], status: ...}`. **Why a flag, not a sentinel.** The record already carries three closed-vocabulary assertions — a real school id asserts *this school applies*, `not-applicable` asserts *the passage advances no theoretical position*, `unlisted` asserts *a real school applies that this vocabulary misses*. Abstention asserts *the draws disagree about which of those is true*: a statement about the draw distribution, not about the passage. Appendix E calls conflating abstention with `not-applicable` "the very error the absence marker exists to prevent", so abstention is kept structurally **outside** the value space and a consumer checks `abstained` before it reads `primary`. An `undecided` token in the vocabulary would have demanded a schema change on every blind axis and let a downstream reader mistake a genuine contest for an asserted category. **Per axis, not per record:** measured abstention rates differ by axis (`theory_school` 8.8% vs `claim_type` 3.3% at N=3) and a chunk routinely decides one blind axis while abstaining the other. **Also decided in the same slice:** a tie for the lead is never broken (coin-flipping a contested chunk is exactly what the marker exists to avoid); a decided axis is byte-identical to the single-draw shape and inherits `secondary`/`subtags` from a draw that actually voted for the winner; a `theory_school` draw soft-landed to `unlisted` casts a **legal ballot**, while a draw still out of vocabulary on a hard-error axis is a **spoiled ballot the vote ignores** — the P0-6 hard error stands only when *every* draw is invalid, preserving the schema-gap guarantee at the chunk level. Head axes (`field`, `empirical_scope`, `role_in_argument`, `polities_touched`) take their **first draw** unchanged: the `N` draws are already in hand so voting them is free, but DEC-31's gains there are small and nothing has asked for it — a later toggle, not a rebuild. Contract in `specs/PRODUCT.md` **§7.14** (plus §7.1, P0-6, Appendix E, Appendix H). `N` is per-pass config (`llm.votes_by_pass`, default 3 for tag, 1 elsewhere), never hardcoded; `N = 1` is an exact no-op. **Still owed:** best-of-N changes how every source is tagged in production (3× calls, new record shape on two axes) and has **not** been validated on the real corpus — that is the stage-4 frozen re-tag, and the 0.918/0.866 figures remain provisional on the simulated gold set (DEC-32). |
